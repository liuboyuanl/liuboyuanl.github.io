【西瓜书】第二章——线性代数笔记


标量、向量、矩阵和张量


  1. 标量
    • 单独的数
    • 斜体表示
    • 实数标量：$ s \in \mathbb{R} $
    • 自然数标量：$ n \in \mathbb{N} $
  2. 向量
    • 一列数，可通过索引得到单独的数
    • 粗体表示
    • 如果每个元素都属于$\mathbb{R}$，并且该向量有$n$个元素，该向量属于实数集$\mathbb{R}$的$n$次笛卡尔乘积构成的集合
    • $$\mathbf{\mathit{x}} =\begin{bmatrix}x_{1}\x_{2} \…\x_{n}\end{bmatrix}$$
    • 索引集合：$S=\left { 1,3,6 \right } $，写作$\mathbf{\mathit{x_{S}}}$，负号表示补集
  3. 矩阵
    • 二维数组
    • 大写粗体
    • 高度为$m$，宽度为$n$
    • $\mathit{A}_{m,n}$：第$m$行第$n$列元素
    • $f(\mathbf{\mathit{A}})_{i,j}$：函数$f$作用在矩阵第$i$行第$j$列
  4. 张量
    • 超过二维的数组
    • $\mathbf{A}$表示张量
  • 深度学习中允许矩阵和向量相加，$\mathbf{\mathit{C}} = \mathbf{\mathit{A}} + \mathbf{\mathit{b}}$，其中$\mathit{C_{i,j}} = \mathit{A_{i, j}} + \mathit{b_{j}}$

线性相关和生成子空间


  • 线性组合：每个向量乘以对应标量系数之后的和
  • 生成子空间：原始向量线性组合后能抵达的点的集合
  • $\mathbf{\mathit{Ax=b}}$是否有解：确定向量$\mathbf{\mathit{b}}$是否在$\mathbf{\mathit{A}}$的生成子空间中
  • 线性相关：如果一组向量中的任意一个向量都不能表示成其他向量的线性组合，那么这组向量成为线性无关，反之为线性相关
  • 一个列向量线性相关的方阵被称为奇异的（singular）
